Combien de kilos de céréale l'éleveur obtient-il en vendant un mouton mâle ? C'est son pouvoir d'achat céréalier. Quand il chute nettement sous la normale de la saison, c'est le signe d'une crise pastorale qui s'installe : le bétail maigrit, les ventes de détresse affluent, la céréale renchérit.

Signal avancé : 3 marché(s) se trouvent dans une wilaya où le bilan fourrager annonce un déficit (⚠️). C'est là que les termes de l'échange risquent de se dégrader dans les semaines qui viennent — le moment d'anticiper un déstockage plutôt que de le subir.
إشارة مبكرة: 3 سوق في ولاية ينذر ميزانها العلفي بعجز — حيث قد تتدهور شروط التبادل قريبًا.
