Installation

  1. Scannez le QR (ou touchez le lien) — le fichier se télécharge.
  2. Ouvrez le fichier app-release.apk téléchargé.
  3. Si Android le demande, autorisez l'installation depuis cette source, puis « Installer ».

التثبيت

  1. امسح الرمز (أو المس الرابط) — يبدأ تنزيل الملف.
  2. افتح الملف المنزَّل app-release.apk.
  3. إذا طلب أندرويد ذلك، اسمح بالتثبيت من هذا المصدر ثم «تثبيت».
Ce lien pointe toujours vers la dernière version : inutile de revenir ici après chaque mise à jour de l'application, il suffit de re-télécharger avec le même QR. Aucune donnée personnelle n'est collectée — voir la politique de confidentialité. هذا الرابط يشير دائمًا إلى أحدث نسخة: لا حاجة للعودة إلى هنا بعد كل تحديث، يكفي إعادة التنزيل بنفس الرمز. لا تُجمع أي بيانات شخصية — انظر سياسة الخصوصية.
