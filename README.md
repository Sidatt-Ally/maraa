# Maraa · مرعى — Observatoire numérique des pâturages et des ressources pastorales de Mauritanie

*المرصد الرقمي للمراعي والموارد الرعوية في موريتانيا — données pastorales ouvertes*

Ce dépôt publie les **produits** du système Maraa :

- **`data/`** — les données décadaires consommées par l'application
  (état des pâturages par moughataa/commune, pluies observées et prévues,
  eau des mares par radar, cheptel RGE 2024, référentiel administratif) ;
- **`docs/`** — le site public : [présentation](https://sidatt-ally.github.io/maraa/),
  [bulletins décadaires FR/AR](https://sidatt-ally.github.io/maraa/bulletins/),
  [méthodologie](https://sidatt-ally.github.io/maraa/methodologie.html) ;
- **branche `live`** — la pluie des dernières heures, rafraîchie chaque heure.

L'historique complet des données est conservé ici : chaque chiffre publié est
daté et vérifiable. Le **système de production** (pipeline, application) est
propriétaire ; un accès d'audit peut être accordé aux institutions dans un
cadre conventionnel — contact via les
[issues](https://github.com/Sidatt-Ally/maraa/issues).

Citation : **Maraa (2026)**. Bulletin pastoral décadaire, Mauritanie.
